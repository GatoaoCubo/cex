```

              │

    │   DECOMPOSE        │  cex_mission.py → intents

              │
    ┌─────────▼──────────┐
    │   8F RUNNER         │  F1 CONSTRAIN → F2 BECOME → F3 INJECT

    │                     │  F7 GOVERN (gates + retry) → F8 COLLABORATE
    └─────────┬──────────┘
              │
    ┌─────────▼──────────┐
    │   VALIDATE          │  cex_hooks.py → pre/post save
    │   + COMPILE         │  cex_compile.py → .yaml
    │   + COMMIT          │  git auto-commit
    └─────────┬──────────┘
              │
    ┌─────────▼──────────┐
    │   LEARN             │  learning_record → .cex/learning_records/
    │   + SCORE           │  cex_score.py → quality assignment
    │   + FEEDBACK        │  cex_feedback.py → metrics.jsonl
    └─────────┬──────────┘
              │
    ┌─────────▼──────────┐
```
