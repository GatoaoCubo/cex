**LLM Evaluation Metrics**  
Metrics quantify model performance in tasks like translation, summarization, and generation. Key metrics include:

| Metric         | Description                              | When to Use                          |
|----------------|------------------------------------------|--------------------------------------|
| **BLEU**       | Measures n-gram overlap with references  | Machine translation, text generation |
| **ROUGE**      | Focuses on recall of key phrases         | Summarization, text compression      |
| **BERTScore**  | Uses semantic similarity via BERT        | Understanding, semantic tasks        |
| **Human Eval** | Subjective assessment by annotators      | Quality judgment, critical tasks     |
| **LLM-as-Judge** | Leverages another model to score outputs | Large-scale evaluation, efficiency  |
