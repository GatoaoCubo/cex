# Voice Accessibility Guidelines | Voice Technology

**Domain**: voice_technology
**Type**: concept
**Quality**: 0.85

Diretrizes WCAG para interfaces de voz.

## Requisitos de Acessibilidade

### Nivel A (Minimo)
- Alternativas de texto para audio
- Controles de reproducao
- Contraste de audio adequado

### Nivel AA (Recomendado)
- Legendas para conteudo de voz
- Descricoes de audio
- Volume ajustavel
- Velocidade de fala configuravel

### Nivel AAA (Ideal)
- Lingua de sinais
- Transcricoes completas
- Customizacao total

## Boas Praticas
- Velocidade de fala: 120-150 palavras/minuto
- Pausas entre secoes
- Tom de voz claro e consistente
- Evitar jargoes tecnicos
