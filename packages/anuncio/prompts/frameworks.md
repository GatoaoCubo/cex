## 1. STORYBRAND FRAMEWORK

### Overview
Created by Donald Miller. Positions the customer as the hero and the brand as the guide.

### 7-Part Structure

```
1. A CHARACTER (Hero)
   - The customer is the hero, not your brand
   - Identify their wants and desires
   - Example: "Voce quer ganhar massa muscular..."

2. HAS A PROBLEM (Villain)
   - External: Tangible obstacle
   - Internal: Emotional frustration
   - Philosophical: Why it's wrong
   - Example: "Mas encontrar um suplemento de qualidade e dificil..."

3. AND MEETS A GUIDE (Your Brand)
   - Show empathy: "We understand"
   - Show authority: "We have experience"
   - Example: "Nossa equipe de nutricionistas entende sua frustracao..."

4. WHO GIVES THEM A PLAN
   - Simple steps (3 is ideal)
   - Clear and actionable
   - Example: "1. Escolha, 2. Compre, 3. Transforme"

5. AND CALLS THEM TO ACTION
   - Direct CTA: "Compre agora"
   - Transitional CTA: "Saiba mais"
   - Example: "Adicione ao carrinho e comece sua transformacao"

6. THAT HELPS THEM AVOID FAILURE
   - Paint what happens if they don't act
   - Create urgency without manipulation
   - Example: "Cada dia sem suplementacao adequada..."

7. AND ENDS IN SUCCESS
   - Show the transformation
   - Paint the desired outcome
   - Example: "Imagine-se com o fisico dos seus sonhos..."
```

### Application in anuncio_agent

| Section | Where Used | Chars |
|---------|------------|-------|
| Hero | Description intro | 400-500 |
| Problem | Description section 2 | 400-500 |
| Guide | Description section 3 | 400-500 |
| Plan | Description section 4 | 300-400 |
| CTA | Description section 5 | 200-300 |
| Failure | Description section 6 | 300-400 |
| Success | Description section 7 | 400-500 |

## 2. AIDA FRAMEWORK

### Overview
Classic marketing funnel: Attention -> Interest -> Desire -> Action

### Structure

```
A - ATTENTION
   Goal: Capture attention in 3 seconds
   Techniques:
   - Impactful headline
   - Specific number
   - Provocative question
   - Bold claim (with evidence)

I - INTEREST
   Goal: Keep them engaged
   Techniques:
   - Storytelling
   - Relevant data
   - Clear benefits
   - Address pain points

D - DESIRE
   Goal: Create wanting
   Techniques:
   - Social proof
   - Visualization of results
   - Exclusivity
   - Scarcity (if real)

A - ACTION
   Goal: Convert
   Techniques:
   - Clear CTA
   - Urgency (if real)
   - Easy next step
   - Risk reversal (guarantee)
```

### Application in anuncio_agent

| Stage | Element | Example |
|-------|---------|---------|
| Attention | Title A | "Whey Protein Isolado 27g Proteina Premium" |
| Interest | Bullets 1-4 | Features transformed to benefits |
| Desire | Bullets 5-8 | Social proof, exclusivity |
| Action | Bullets 9-10, Description CTA | "Compre agora" |

## 3. PAS FRAMEWORK

### Overview
Problem -> Agitate -> Solution

### Structure

```
P - PROBLEM
   Identify the specific problem
   Use customer's own words
   Example: "Dificil encontrar whey protein de qualidade"

A - AGITATE
   Amplify the pain
   Show consequences
   Make them feel the problem
   Example: "Cada dia com suplemento ruim e dinheiro jogado fora"

S - SOLUTION
   Present your product as THE answer
   Show transformation
   Example: "Whey Protein Isolado resolve com 27g de proteina pura"
```

### Application in anuncio_agent

Used in individual bullets and description sections.

## 4. FAB FRAMEWORK

### Overview
Feature -> Advantage -> Benefit

### Structure

```
FEATURE: What the product HAS
Example: "27g de proteina isolada"

ADVANTAGE: What it DOES better
Example: "90% de absorcao em 30 minutos"

BENEFIT: What the customer GETS
Example: "Maximo ganho muscular pos-treino"

Complete formula:
"27g de proteina isolada (feature) com 90% de absorcao em 30 minutos (advantage) para maximo ganho muscular pos-treino (benefit)"
```

### Application in anuncio_agent

Used to structure each bullet point:
```
BENEFIT: Feature que advantage para resultado
```

## 5. 4U FRAMEWORK

### Overview
For headlines and titles: Useful, Urgent, Unique, Ultra-specific

### Structure

```
USEFUL: Clear benefit
URGENT: Reason to act now
UNIQUE: Differentiation
ULTRA-SPECIFIC: Precise details

Example (title):
"Whey Protein Isolado 27g Proteina 90% Pureza"
- Useful: Proteina para ganho muscular
- Unique: 27g por dose (especifico)
- Ultra-specific: 90% pureza
```

### Application in anuncio_agent

Primary framework for title generation (Step 2).

## 6. MENTAL TRIGGERS

### Quick Reference

| Trigger | Description | Use Case |
|---------|-------------|----------|
| Social Proof | Others trust it | Bullets, Description |
| Authority | Expert endorsement | Bullets, Description |
| Scarcity | Limited availability | Bullets (if real) |
| Urgency | Time-limited | Description (if real) |
| Reciprocity | Give something | Bullets, Description |
| Specificity | Precise numbers | Titles, Bullets |
| Curiosity | Information gap | Bullets |
| Fear of Loss | What they'll miss | Description |
| Exclusivity | Special access | Bullets |
| Guarantee | Risk reversal | Bullets, Description |

### Recommended Distribution

```
10 Bullets should use:
- Bullet 1: Specificity (numbers)
- Bullet 2: Authority (credibility)
- Bullet 3: Social Proof (trust)
- Bullet 4: Specificity (more numbers)
- Bullet 5: Exclusivity (unique value)
- Bullet 6: Guarantee (risk reversal)
- Bullet 7: Specificity (technical)
- Bullet 8: Curiosity (engagement)
- Bullet 9: Fear of Loss (motivation)
- Bullet 10: Reciprocity (added value)
```

## 7. BRAZILIAN E-COMMERCE SPECIFICS

### Cultural Considerations

```yaml
tone: "Friendly but professional"
formality: "Voce" (not "Tu" or "Senhor")
price_sensitivity: High (emphasize value)
trust_signals: Important (ANVISA, INMETRO)
installments: Always mention "12x sem juros"
```

### Platform-Specific Adaptations

```yaml
mercado_livre:
  - Full feature utilization
  - Rich description important
  - Reputation mentions valuable

shopee:
  - Younger audience
  - Price-focused
  - Hashtags valuable
  - Vouchers/promos important

amazon_br:
  - A+ content when available
  - Prime eligibility
  - Structured bullets

magalu:
  - Local trust important
  - "Retira na loja" mention
  - Parceiro Magalu badge
```

## 8. COMPLIANCE QUICK GUIDE

### ANVISA (Suplementos/Cosmeticos)

```yaml
NUNCA dizer:
  - "Cura" / "Trata"
  - "Previne doenca"
  - "Emagrece X kg"
  - "Elimina gordura"
  - "Resultado garantido"
  - "Sem efeitos colaterais"
  - "100% natural = seguro"

PODE dizer:
  - "Contribui para"
  - "Auxilia na"
  - "Pode ajudar a"
  - "Complementa alimentacao"
```

### CONAR (Publicidade)

```yaml
NUNCA:
  - Publicidade enganosa
  - Comparacao desleal
  - Claims sem prova
  - Preco mentiroso
  - Urgencia falsa
  - Escassez artificial
```

### INMETRO (Produtos)

```yaml
Quando aplicavel:
  - Selo INMETRO visivel
  - Voltagem especificada
  - Certificacao mencionada
```
