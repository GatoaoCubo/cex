The Capability Registry is a searchable catalog of all agents available to crews. It indexes 252 builder sub-agents, nucleus domain agents, and nucleus cards. Key fields include:

- **capability_name**: Unique identifier for the agent's function
- **provider_agent**: Core agent responsible for execution
- **input/output schemas**: Formalized data structures for interaction
- **cost**: Estimated resource consumption metric
- **quality_baseline**: Minimum performance threshold
- **availability**: Operational status and access constraints

This structured metadata enables ranked candidate discovery, allowing crews to select optimal agents based on technical specifications and resource constraints.
