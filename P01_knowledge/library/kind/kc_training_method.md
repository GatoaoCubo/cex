Training Method refers to techniques used to train or adapt machine learning models. Key methods include:

- **Supervised Learning**: Using labeled data to train models
- **Unsupervised Learning**: Finding patterns in unlabeled data
- **Reinforcement Learning**: Training via reward signals
- **Transfer Learning**: Leveraging pre-trained models
- **Fine-tuning**: Adjusting pre-trained models for specific tasks
- **Model Pruning**: Removing redundant parameters
- **Quantization**: Reducing model size via numerical precision
- **Knowledge Distillation**: Transferring knowledge from large to small models

Evaluation considers accuracy, efficiency, and generalization. Choosing the right method depends on data, task, and resource constraints.
