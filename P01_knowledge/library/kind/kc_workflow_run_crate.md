The RO-Crate 1.2 Workflow Run Crate is a standardized format for capturing scientific workflow execution provenance. It enables explicit tracking of CreateAction metadata, including ORCID attribution for contributors, while ensuring FAIR (Findable, Accessible, Interoperable, Reusable) principles through structured metadata.

Key features:
- Provenance tracking of computational workflows
- ORCID integration for author attribution
- FAIR metadata compliance
- CreateAction semantic enrichment
- Versioned workflow execution records
- Interoperable with research data ecosystems

This crate format supports reproducible science by providing persistent identifiers for workflows, input datasets, and output artifacts, while maintaining metadata quality and discoverability across disciplines.
