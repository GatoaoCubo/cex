# Enterprise SLA Template

**Uptime Guarantee**  
99.9% monthly uptime (excluding scheduled maintenance)  
99.95% quarterly uptime (excluding scheduled maintenance)  
99.99% annual uptime (excluding scheduled maintenance)

**Latency Commitments**  
Average response time: < 150ms for critical requests  
Maximum latency: < 300ms for all requests  
99th percentile latency: < 250ms

**Support Commitments**  
24/7 technical support with 1-hour response time  
Priority support for enterprise customers  
SLA monitoring dashboard with real-time metrics  
Service credits for downtime exceeding 99.9% guarantee  
Quarterly service reviews with performance reports  
Custom SLA configurations for mission-critical applications
