# SEO Content Strategy for E-commerce Sites


## Key Concepts
- **Commercial Keywords**: Buy/purchase intent terms with CPC >$2, conversion rate 3-8%
- **Product Schema**: Structured data for price, availability, reviews (rich snippets boost CTR 20-30%)
- **Content Hierarchy**: Homepage → Category → Subcategory → Product (max 3 clicks from home)
- **Faceted Navigation**: SEO-friendly filters without duplicate content or infinite URL parameters

## Strategy Phases
1. **Audit**: Identify commercial keyword gaps, technical issues, content thin pages
2. **Research**: Map buyer journey keywords (awareness → consideration → purchase)
3. **Create**: Product descriptions, category content, buying guides, comparison pages
4. **Optimize**: Title tags, meta descriptions, internal linking, page speed <3s
5. **Monitor**: Track organic traffic, conversion rate, average order value from SEO

## Golden Rules
- TARGET long-tail product keywords (3+ words, lower competition, higher intent)
- STRUCTURE URLs as /category/subcategory/product-name (breadcrumb hierarchy)
- WRITE unique descriptions >150 words per product (avoid manufacturer copy)
- IMPLEMENT review schema and FAQ markup for enhanced SERP features
- OPTIMIZE for mobile-first indexing with responsive design and fast loading
