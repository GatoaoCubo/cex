# Multimodal Prompt Patterns

Cross-modal prompts integrate vision, audio, and text to create richer context for AI systems. This pattern enables simultaneous processing of multiple modalities to enhance understanding and output quality.

## Vision Modality
- Describe visual elements and link them to text
- Example: "Describe the visual elements in this image and explain how they relate to the text provided"

## Audio Modality
- Transcribe and analyze sound, then connect to text
- Example: "Transcribe the audio clip and analyze how the sounds relate to the text content"

## Text Modality
- Standard text prompts with contextual framing
- Example: "Analyze this text for key themes and connect them to the visual/audio elements"

## Combined Patterns
1. "Analyze the image, transcribe the audio, and then create a cohesive narrative integrating all elements"
2. "Describe the visual elements, analyze the audio content, and then explain how they collectively support the text's message"

Benefits:
- Enhanced context understanding
- Richer output generation
- Better cross-modal reasoning
- Improved task completion for complex multimodal tasks
