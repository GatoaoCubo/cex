# Churn Prevention Playbook

## Signal Detection
- Monitor usage patterns (e.g., 30% drop in activity)
- Track support ticket volume (e.g., >5 unresolved tickets)
- Analyze engagement metrics (e.g., 50% reduction in logins)
- Flag account inactivity (e.g., >90 days without login)

## Intervention Triggers
- **Automated alerts**: Send personalized email with usage stats
- **Manual review**: Flag accounts with high support needs
- **Usage thresholds**: Trigger re-engagement campaigns
- **Payment issues**: Alert for failed payments or subscription gaps

## Save-the-Account Scripts
1. **Welcome back**: Offer 20% discount on next renewal
2. **Account review**: Schedule 1:1 session to address concerns
3. **Loyalty reward**: Grant premium features for 3 months
4. **Churn warning**: Send final notice with exit survey

## Key Metrics
- Churn rate (monthly)
- Recovery rate (post-intervention)
- Customer satisfaction score
- Cost per retention action

## Best Practices
- Act within 7 days of signal detection
- Personalize communication with account-specific data
- Document all interventions for audit purposes
- Continuously refine triggers based on historical data
