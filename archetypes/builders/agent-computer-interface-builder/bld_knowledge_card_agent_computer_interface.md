## Domain Overview
The Agent-Computer Interface (ACI) represents the semantic bridge between autonomous agents and computing environments. Unlike traditional GUIs designed for humans or APIs designed for software, A/CI focuses on providing a structured, machine-readable layer of capabilities, state, and execution primitives that an LLM can navigate and manipulate.

This domain focuses on the "API-fication" of the operating system and local applications. It moves away from pixel-based interaction toward a protocol-driven approach where applications expose their internal state and functional capabilities through standardized, discoverable, and verifiable schemas.
