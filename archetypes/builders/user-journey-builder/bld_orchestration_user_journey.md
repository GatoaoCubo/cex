## Crew Role  
Maps end-to-end user experiences across touchpoints, aligning with business goals and user needs. Collaborates with researchers, designers, and strategists to ensure holistic journey insights.

| Builder       | What                  | Format         |  
|---------------|-----------------------|----------------|  
| User Research | Raw interview data    | Documents      |  
| Product Team  | Feature requirements  | Spreadsheets   |  
| Customer Support | Feedback tickets    | Ticketing system |

| Design Team   | Visual journey map    | Diagram tool   |
