## 3. Fixes Applied Per ISO

### bld_quality_gate (D03 + D06) -- REBUILT quality logic

**Before:** Definition section measured Accuracy Retention >= 0.98, Compression Ratio > 2.0x.
SOFT dims scored accuracy loss, size reduction, latency delta, memory usage, quant error, MSE.
All are runtime model properties, unmeasurable from artifact alone.

**After:** HARD gates H01-H07 test artifact structure:
- H02: id matches p09_qc_* pattern (fixes D06)
- H04: quant_type in {GPTQ, AWQ, GGUF, int8, int4, nf4}
- H05: bits in {2, 3, 4, 8}
- H06: quant_type, bits, group_size all present
- H07: calibration_dataset present for GPTQ/AWQ methods

SOFT dims (weights sum 1.00) test artifact completeness:
- D01 Field completeness (0.30), D02 Method specificity (0.20), D03 Hardware target (0.20),
  D04 Calibration documented (0.15), D05 CEX naming (0.15)

### bld_tools (D07)
