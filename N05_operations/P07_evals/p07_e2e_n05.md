## Pipeline Overview

```
[brand_init.yaml]
      |
[bootstrap] -> brand_config.yaml + propagate
      |
[build_8f] -> knowledge_card .md + .yaml
      |
[dispatch_solo] -> handoff -> nucleus boot -> artifact -> signal
      |
[grid_6] -> 6 nuclei -> 6 signals -> 6 deliverables
      |
[compile] -> .md -> cex_compile.py -> .yaml sidecar
      |
[doctor] -> cex_doctor.py -> exit 0
      |
[signal] -> signal_writer.py -> JSON signal file
      |
[quality_gate] -> cex_score.py -> gate pass -> cex_feedback.py
```
