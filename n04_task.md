# HL7 FHIR R5 AI Agent Capability Declaration

This capability declaration defines requirements for AI agents interoperating with FHIR R5 systems. It establishes:

1. **SMART on FHIR Scopes**  
   - Read/Write access to patient records  
   - Clinical decision support permissions  
   - System administration capabilities  

2. **PHI Handling**  
   - Explicit patient consent requirements  
   - Anonymization protocols for secondary use  
   - Audit trail for data access  

3. **CDS Hooks Integration**  
   - Pre-authorization for clinical interventions  
   - Real-time decision support triggers  
   - Evidence-based recommendation validation  

4. **AI Transparency**  
   - Model provenance documentation  
   - Bias mitigation strategies  
   - Explainability frameworks for clinical decisions  

This declaration enables secure, compliant interactions with FHIR-based health systems while maintaining patient data integrity and regulatory compliance.
