description: |
  This knowledge card defines the standardized metadata format for marketplace app listings across Claude, LangChain, and HuggingFace platforms. The manifest includes:
  
  1. **Metadata**: App name, description, version, license type, and category
  2. **Permissions**: Access controls for data processing, API integrations, and user interactions
  3. **Pricing**: Subscription models, pay-per-use rates, and discount structures
  4. **Platform Compatibility**: Specific configuration parameters for Claude (anthropic), LangChain (llm), and HuggingFace (transformers) integrations
  5. **Security**: Encryption standards, authentication protocols, and data privacy policies
  6. **Dependencies**: Required libraries, framework versions, and environment variables
  7. **Performance Metrics**: Expected response times, throughput benchmarks, and resource requirements
  
  This spec enables consistent app discovery, trust verification, and automated deployment across all supported platforms.
