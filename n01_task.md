description: |
  This knowledge card defines the metadata structure for marketplace applications listing on Claude, LangChain, and HuggingFace platforms. The manifest includes:
  - Core metadata (name, description, version)
  - Permission declarations (data access, API integrations)
  - Pricing model specifications (freemium, subscription, pay-per-use)
  - Platform-specific configuration (runtime dependencies, environment variables)
  - Security requirements (encryption standards, authentication methods)
  - Compliance declarations (GDPR, CCPA, etc.)
  - Developer contact information
  - License terms and conditions
