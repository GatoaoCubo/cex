## Key Concepts
- **Efficiency**: Condenses information into high-density format.
- **Relevance**: Prioritizes essential facts for decision-making.
- **Retrievability**: Simplifies access to important data rapidly.

## Strategy Phases
1. **Identification**: Select core information from broader content.
2. **Compression**: Distill information into concise, actionable points.
3. **Verification**: Ensure extracted data accuracy and relevance.

## Golden Rules
- PRIORITIZE essential over supplementary information.
- MAXIMIZE density without losing context or meaning.
- VALIDATE accuracy before integrating into systems.

## Flow
```
[Input Data] -> [Identify Core] -> [Compress] -> [Verify] -> [Knowledge Nucleus]
```

## Comparativo
