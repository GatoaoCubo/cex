# Template: Blog Listicle Post

> For numbered list posts. 800-1200 words. Voice: Ro (sofisticado-acolhedor).
> High shareability, scannable, SEO-friendly.

## Post Structure

```
1. TITLE (H1)
2. META DESCRIPTION
3. HERO IMAGE
4. INTRODUCTION (80-120 words)
5. LIST ITEMS (H2 per item, 5-10 items)
6. BONUS ITEM (optional, H2)
7. CONCLUSION + CTA (80-100 words)
8. RELATED PRODUCTS
```

## Section Details

### 1. Title (H1)

| Formula | Example |
|---------|---------|
| "[N] [Noun] que [verb/benefit]" | "7 Brinquedos que Todo Gato Precisa Ter" |
| "[N] [Adjective] [Noun] para [audience]" | "10 Presentes Perfeitos para Maes de Gato" |
| "Os [N] Melhores [Noun] de [year]" | "As 5 Melhores Camas para Gatos de 2026" |
| "[N] [Noun] que Voce Nao Sabia sobre [topic]" | "8 Fatos que Voce Nao Sabia sobre Sono Felino" |

**Rules:**
- Always include the number
- Max 60 chars for SEO title
- Odd numbers (5, 7, 9) perform slightly better than even

### 2. Introduction (80-120 words)

```
[Hook -- 1 sentence that creates curiosity or relates to ICP]
"Se voce ja se perguntou [common question], essa lista e pra voce."

[Context -- why this list matters]
"A Ro selecionou [N] [items] que fazem a diferenca real no dia a dia."

[What to expect]
"De [first item type] a [last item type], cada um foi escolhido com criterio."
```

Keep it tight. Listicles are about scanning -- get to the list fast.

### 3. List Items (H2 per item)

Each item follows this mini-structure:

```
## [N]. [Item Title]

[1 paragraph -- 60-100 words explaining the item]

**Por que funciona:** [1 sentence -- the science or logic]
**Dica da Ro:** [1 practical sentence]

[Optional: product link if item is/relates to a product]
```

**Item writing rules:**
- **Consistent structure** across ALL items (same sub-sections)
- **Front-load value** -- best items at positions 1, 2, and last
- **Vary item types** -- mix facts, tips, products, myths
- Each item must stand alone (reader may scan, not read all)
- Product mentions in max 40% of items (not every single one)

### 4. Bonus Item (Optional)

```
## Bonus: [Item Title]

"E um extra que a Ro nao podia deixar de fora..."
[Same structure as regular items]
```

Bonus item creates delight and over-delivery. Use for product showcase.

### 5. Conclusion + CTA (80-100 words)

```
## Conclusao

[1-2 sentence recap]
"Esses [N] [items] cobrem o essencial para [benefit]."

[Engagement CTA]
"Qual desses voce ja conhecia? Comenta la no nosso Instagram @gatoaocubo3."

[Next step CTA]
"Quer se aprofundar? Leia tambem: [related how-to post link]."
```

## SEO Checklist

- [ ] Number in title and H1
- [ ] Primary keyword in title, meta, first 100 words
- [ ] Each H2 item contains a variation of the keyword
- [ ] Internal links: 2+ product pages inline, 1+ related blog post
- [ ] URL: /blog/[n]-[keyword-slug]
- [ ] FAQ schema if items are Q&A format
- [ ] Images: 1 hero + 1 per item if product listicle
- [ ] Alt text with keyword variations

## Listicle Types Matrix

| Type | Item count | Word count | Product integration | Example |
|------|-----------|-----------|-------------------|---------|
| Curated products | 5-10 | 1000-1200 | High (score 3) | "7 melhores brinquedos interativos" |
| Tips/dicas | 5-10 | 800-1000 | Medium (score 1-2) | "10 dicas para o primeiro dia com gato" |
| Facts/fatos | 5-8 | 800-1000 | Low (score 0-1) | "8 fatos sobre sono felino" |
| Mistakes/erros | 5-7 | 800-1000 | Medium (score 1) | "5 erros que tutores iniciantes cometem" |
| Myths/mitos | 5-8 | 1000-1200 | Low (score 0-1) | "7 mitos sobre gatos que voce ainda acredita" |

## Brand Voice for Listicles

- **Tone:** lighter than how-to, more conversational
- **Ro's presence:** intro + 3-4 "Dica da Ro" throughout
- **Humor allowed:** mild, warm humor in meme-adjacent items
- **Product voice:** "a gente recomenda" or "a curadoria GATO3 inclui"
- **Never:** rank competitors, guarantee results, use superlatives without evidence
