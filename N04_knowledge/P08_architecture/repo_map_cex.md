## Key Entry Points (for new contributors)

| Goal | Start here |
|------|-----------|
| Add a new builder | `CONTRIBUTING.md` Path 1 -> copy from `archetypes/builders/agent-builder/` |
| Add a knowledge card | `N00_genesis/P01_knowledge/library/kind/` + `kc_{topic}.md` |
| Add an SDK provider | `cex_sdk/models/` + `.cex/config/nucleus_models.yaml` |
| Add a vertical nucleus | `CONTRIBUTING.md` Path 4 -> 5 required files |
| Run full health check | `python _tools/cex_doctor.py` |
| Run a builder locally | `python _tools/cex_8f_runner.py --kind {kind}` |
| Dispatch a nucleus | `bash _spawn/dispatch.sh solo n04 "task"` |
| Run a crew | `python _tools/cex_crew.py run {crew_name} --execute` |
| Check artifact quality | `python _tools/cex_score.py {file_path}` |
