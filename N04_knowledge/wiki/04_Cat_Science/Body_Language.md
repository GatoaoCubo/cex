# Body Language

Guia completo de linguagem corporal felina. Base para carrosseis educativos no [[Pillar_Educativo]].

## Cauda (Tail Decoder)

| Posicao | Significado | Angulo de conteudo |
|---------|------------|-------------------|
| Reta para cima | Confiante, feliz | "Seu gato esta feliz? Olha o rabo!" |
| Curvada (interrogacao) | Brincalhao, curioso | "Hora de brincar!" |
| Agitando forte | Superestimulado, irritado | "PARE: sinais de que quer espaco" |
| Arrepiada (bottle brush) | Medo, fight-or-flight | "Emergencia: gato em panico" |
| Baixa/presa ao corpo | Submissao, medo, doenca | "5 sinais de que algo esta errado" |
| Enrolada no corpo | Calmo, auto-conforto | "O abraco felino" |

## Orelhas (Ear Decoder)

| Posicao | Significado | Angulo |
|---------|------------|--------|
| Para frente, eretas | Neutro, relaxado | Referencia baseline |
| "Orelhas de aviao" (planas laterais) | Ansiedade, estresse subindo | "Hora de recuar" |
| Coladas para tras | Medo, agressao defensiva | "NUNCA force carinho assim" |
| Girando independentemente | Rastreando sons, vigilante | "Radar felino" (Reel divertido) |

## Slow Blink (Beijo de Gato)

O slow blink e um sinal de confianca -- equivalente a um "beijo" na comunicacao felina. Gatos usam para sinalizar nao-ameaca. Retribuir o slow blink constroi confianca.

**Conteudo**: "Voce sabe fazer o 'beijo de gato'?" -- Reel demonstrando tecnica (alta shareability). Top 10 posts ([[Pillar_Educativo]]).
