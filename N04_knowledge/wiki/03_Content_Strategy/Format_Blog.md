# Format Guide -- Blog

Blog em gato3.com.br/blog. 2 posts/semana. Foco em SEO + autoridade.

## Formatos

| Formato | Palavras | Estrutura | Quando |
|---------|---------|-----------|--------|
| How-to | 1500-2000 | Intro + Steps + Conclusao + CTA | Ensinar um processo |
| Listicle | 800-1200 | Intro + Itens numerados + CTA | Listas curadas, dicas |
| Deep dive | 2000-3000 | Intro + Secoes + Dados + Conclusao | Autoridade, SEO pillars |
| Quick tip | 500-800 | Problema + Solucao + Produto | Respostas rapidas, long-tail SEO |

## Regras de SEO

Cada blog post deve:
1. Keyword primaria no titulo/H1/meta/primeiras 100 palavras/URL
2. 2-3 long-tail keywords no corpo
3. FAQ schema quando aplicavel
4. 2+ links internos (para produtos + outro blog post)
5. Alt text em todas as imagens

## Linking Interno

Todo post deve linkar para:
- Pelo menos 2 paginas de [[Product_Index|produto]]
- Pelo menos 1 outro blog post
- CTA para Instagram (@gatoaocubo3)

## Avaliacao

D4 da [[Content_Quality_Rubric]] mede otimizacao SEO. Peso: 15%. So aplica para blog (N/A em Instagram).

## Modelo de Blog

O [[Tuft_Paw]] (tuftandpaw.com/blogs/cat-guides) e a referencia de SEO-driven cat content. "Definitive Guide to Cat Body Language" ranqueia #1 para multiplas keywords. GATO3 deve replicar essa estrategia em PT-BR onde a concorrencia e quase zero.

## Competicao SEO em PT-BR

Praticamente nenhum site brasileiro tem cat behavior content com [[Tier_1_Academic|fontes Tier 1]] em portugues. Isso e blue ocean para gato3.com.br.
