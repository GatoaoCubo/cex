## Regras

1. **Total por post**: 20-25 hashtags (sweet spot Instagram 2026)
2. **Mix**: 5 marca + 10 categoria + 5-10 trending/nicho
3. **Colocacao**: primeiro comentario (NAO na legenda) -- manter caption limpa
4. **Rotacao**: trocar 5 trending tags por semana para evitar shadowban
5. **Pesquisa**: revisao semanal via Instagram Insights

## Anti-Patterns

- NAO usar 30 hashtags (shadowban risk)
- NAO repetir exatamente o mesmo set toda semana
- NAO usar tags irrelevantes para buscar alcance
