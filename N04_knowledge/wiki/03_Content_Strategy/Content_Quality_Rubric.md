# Content Quality Rubric

Score minimo: **4.0/5.0**. Qualquer dimensao abaixo de 3.0 = rejeicao automatica.

## 6 Dimensoes

### D1 -- Brand Voice Alignment (20-25%)

Conteudo combina com [[Brand_Voice|tom sofisticado-acolhedor]]?

| 5 | Voz perfeita da [[Persona_Ro|Ro]]. Autoridade calorosa, ciencia acessivel. |
| 4 | Match forte. Desvios menores em 1-2 frases. |
| 3 | Aceitavel mas generico. Poderia ser qualquer marca pet. |
| 2 | Mismatch de voz. Clinico, vendedor, ou condescendente demais. |
| 1 | Voz errada. CAPS LOCK, hard sell, emoji overload. |

**Red flags** (auto-fail): "COMPRE AGORA", infantilizar tutor, claims medicos sem vet.

### D2 -- Visual Consistency (15%)

Combina com [[Visual_Identity|estetica PB minimalista]]?

| 5 | PB perfeito, tipografia Allrounder/Kenao, layout limpo. |
| 3 | Aceitavel mas nao distintivo. Look generico. |
| 1 | Off-brand. Cores vibrantes, clip art, stock com watermark. |

### D3 -- CTA Clarity (15%)

CTA claro e unico alinhado com o tipo de conteudo?

| Tipo | CTA primario | CTA secundario |
|------|-------------|----------------|
| [[Pillar_Educativo]] | "Salva pra depois" | "Comenta sua duvida" |
| [[Pillar_Humor]] | "Marca alguem" | "Compartilha" |
| [[Pillar_Produto]] | "Link na bio" | "Salva pra quando precisar" |
| [[Format_Blog\|Blog how-to]] | "Leia o guia completo" | "Veja produtos relacionados" |
| Story | "Vote/Responda" | "Swipe up" |

### D4 -- SEO Optimization (15%, Blog only)

Keyword no titulo, H1, meta, primeiras 100 palavras? Links internos? Schema?
Para Instagram: N/A (peso redistribuido para D1 e D5).

### D5 -- Engagement Potential (20-25%)

Shareable? Saveable? Provoca comentarios?

Sinais: hook strong? Ensinaria algo para salvar? Mandaria para amigo gateiro?

### D6 -- Product Integration Naturalness (15-20%)

Produto aparece como solucao natural? Leitor nao se sente "vendido"?

| Score 0 (sem produto) | = Score 5 (decisao correta) |
| Score 1 (sutil) | = Score 4-5 |
| Score 3 (product-first) | = Score 3-4 (dificil ser 5) |

## Formula

**Blog**: `(D1*0.20) + (D2*0.15) + (D3*0.15) + (D4*0.15) + (D5*0.20) + (D6*0.15)`

**Instagram**: `(D1*0.25) + (D2*0.15) + (D3*0.15) + (D5*0.25) + (D6*0.20)`

## Gates

| Gate | Condicao | Acao |
|------|----------|------|
| Hard reject | Qualquer D < 3.0 | Reescrever |
| Revise | Media < 4.0 | Revisar dimensoes fracas |
| Approve | Media >= 4.0 E nenhum D < 3.0 | Agendar |
| Excellence | Media >= 4.5 | Boost/promover |
