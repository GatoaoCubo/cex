# Template -- Blog Post

Use esta estrutura para cada post no gato3.com.br/blog.

## Checklist Pre-Escrita

- [ ] Keyword primaria definida
- [ ] 2-3 long-tail keywords identificadas
- [ ] Formato escolhido (how-to, listicle, deep dive, quick tip)
- [ ] Fontes [[Source_Index|verificadas]] (minimo Tier 2)
- [ ] Produtos para bridge identificados ([[Pillar_Produto]])

## Estrutura Padrao

```markdown
# [Titulo com keyword primaria]

> [Meta description: 155 chars, keyword, CTA implicito]

## Introducao (100-150 palavras)
- Hook: pergunta ou dado surpreendente
- Contexto: por que isso importa
- Preview: o que o leitor vai aprender

## [Secao 1: Problema/Contexto]
- Dados cientificos com fonte
- Citacao Tier 1 quando possivel

## [Secao 2-4: Conteudo principal]
- Passos, items, ou analise
- Tabelas e listas para scannability
- Imagens com alt text descritivo

## [Secao 5: Ponte para produto] (ultimos 20%)
- Transicao natural: "Uma solucao pratica e..."
- Produto como exemplo, nao como venda
- Link para pagina do produto

## Conclusao
- Resumo em 2-3 frases
- CTA claro (salvar, compartilhar, ver produtos)

## FAQ (se aplicavel)
- 2-3 perguntas em formato schema
```

## Regras [[Brand_Voice|de Voz]]

- Tom [[Persona_Ro|da Ro]]: acolhedora, informativa, parceira
- Frases curtas, paragrafos enxutos
- "Voce" direto ao tutor
- Sem CAPS, sem exclamacoes excessivas

## SEO Checklist ([[Content_Quality_Rubric|D4]])

- [ ] Keyword no titulo/H1
- [ ] Keyword nas primeiras 100 palavras
- [ ] Keyword na URL slug
- [ ] 2+ links internos (produto + outro post)
- [ ] Alt text em todas imagens
- [ ] Meta description com keyword
