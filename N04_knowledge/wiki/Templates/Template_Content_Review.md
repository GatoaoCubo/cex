# Template -- Content Review

Ficha de avaliacao baseada na [[Content_Quality_Rubric|rubrica 6D]].

## Ficha de Scoring

```
CONTEUDO: _____________________________
DATA: ____________  REVIEWER: __________
PLATAFORMA: [ ] Instagram  [ ] Blog

D1 Voice ([[Brand_Voice]]):       [ 1 ] [ 2 ] [ 3 ] [ 4 ] [ 5 ]  x0.20 = ___
D2 Visual ([[Visual_Identity]]):  [ 1 ] [ 2 ] [ 3 ] [ 4 ] [ 5 ]  x0.15 = ___
D3 CTA:                           [ 1 ] [ 2 ] [ 3 ] [ 4 ] [ 5 ]  x0.15 = ___
D4 SEO ([[Format_Blog]]):        [ 1 ] [ 2 ] [ 3 ] [ 4 ] [ 5 ]  x0.15 = ___ (N/A for IG)
D5 Engagement:                    [ 1 ] [ 2 ] [ 3 ] [ 4 ] [ 5 ]  x0.20 = ___
D6 Product ([[Pillar_Produto]]):  [ 1 ] [ 2 ] [ 3 ] [ 4 ] [ 5 ]  x0.15 = ___

TOTAL: ___/5.0

GATE: [ ] Approve (>=4.0)  [ ] Revise (<4.0)  [ ] Reject (any D<3.0)  [ ] Excellence (>=4.5)

NOTAS:
_________________________________________________
```

## Pesos (Instagram vs Blog)

| Dimensao | Blog | Instagram |
|----------|------|-----------|
| D1 Voice | 20% | **25%** |
| D2 Visual | 15% | 15% |
| D3 CTA | 15% | 15% |
| D4 SEO | 15% | **N/A** |
| D5 Engagement | 20% | **25%** |
| D6 Product | 15% | **20%** |

## Gates

| Gate | Condicao | Acao |
|------|----------|------|
| **Hard reject** | Qualquer D < 3.0 | Reescrever a dimensao, re-score |
| **Revise** | Media < 4.0 | Revisar dimensoes fracas |
| **Approve** | Media >= 4.0 E nenhum D < 3.0 | Agendar |
| **Excellence** | Media >= 4.5 | Boost / promover |
