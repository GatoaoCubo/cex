# Template -- Instagram Post

Checklist de publicacao para cada post Instagram.

## Checklist Pre-Publicacao

### Conteudo
- [ ] Pilar definido: [[Pillar_Educativo|Edu]] / [[Pillar_Humor|Humor]] / [[Pillar_Produto|Produto]] / [[Pillar_Seasonal|Sazonal]]
- [ ] Formato escolhido ([[Format_Instagram]])
- [ ] Hook nos primeiros 125 chars da caption
- [ ] CTA unico e claro
- [ ] Score de integracao produto: 0-3

### Visual ([[Visual_Identity]])
- [ ] Paleta PB (preto, branco, cinza)
- [ ] Tipografia Allrounder (titulos) / Kenao (corpo)
- [ ] Resolucao correta (1080x1080 ou 1080x1350)
- [ ] Watermark GATO3 (feed)

### Caption
- [ ] Tom [[Brand_Voice|sofisticado-acolhedor]]
- [ ] Voz da [[Persona_Ro|Ro]] (se educativo)
- [ ] Sem CAPS LOCK, sem emoji overload
- [ ] Sem comparacao com concorrentes por nome

### Hashtags ([[Hashtag_Strategy]])
- [ ] 20-25 tags total
- [ ] 5 tags de marca
- [ ] 10 tags de categoria
- [ ] 5-10 trending/nicho
- [ ] No primeiro comentario (nao na caption)

### Final
- [ ] Avaliado pela [[Content_Quality_Rubric|rubrica 6D]]
- [ ] Score >= 4.0/5.0
- [ ] Nenhuma dimensao < 3.0
- [ ] Agendado no calendario ([[Editorial_Calendar_Q2]])

## Estrutura de Caption

```
[HOOK - primeira frase que para o scroll]

[CONTEUDO - 3-5 frases com valor]

[CTA - acao unica e clara]

[ASSINATURA - "Com carinho, Ro" ou similar quando aplicavel]
```
