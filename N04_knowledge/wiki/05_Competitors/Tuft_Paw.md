# Tuft+Paw

Premium modern cat furniture. ~35 funcionarios. Blog = masterclass em SEO de conteudo pet.

## Profile

| Dimensao | Avaliacao |
|----------|----------|
| Fundacao | ~2017 |
| Posicionamento | Premium modern cat furniture |
| Funcionarios | ~35 (3 continentes, 2026) |
| Crescimento | PR-driven + email marketing (Klaviyo) |
| Contratacoes | Social media (YouTube/TikTok priority) |

## Forcas

- Posicionamento premium mais forte em cat furniture
- Blog excelente (guias completos de comportamento)
- Publico design-conscious (overlap com [[ICP_Profile|ICP GATO3]])
- PR como motor de crescimento

## Fraquezas (Oportunidades GATO3)

- **Preco exclui mercado BR** -- ICP GATO3 (B-C+) nao pode pagar T+P
- **Shipping/mercado NA** -- sem presenca no Brasil
- **Moveis apenas** -- GATO3 tem catalogo mais amplo

## Aprendizado-Chave para GATO3

O **blog** (tuftandpaw.com/blogs/cat-guides) e uma masterclass em SEO cat content. "Definitive Guide to Cat Body Language" ranqueia #1 para multiplas keywords.

GATO3 deve replicar essa estrategia para gato3.com.br/blog. Em PT-BR a concorrencia e **quase zero** para cat behavior content com fontes cientificas. Ver [[Format_Blog]].
