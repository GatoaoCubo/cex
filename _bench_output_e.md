# Agent Memory Architectures

## 4 Memory Types

1. **Working Memory**  
   - Temporary storage for current tasks  
   - Used: For real-time decision-making and active problem-solving

2. **Episodic Memory**  
   - Stores personal experiences and events  
   - Used: To recall specific events or lessons learned

3. **Semantic Memory**  
   - Retains general knowledge and facts  
   - Used: For understanding concepts and contextual information

4. **Procedural Memory**  
   - Encodes skills and routines  
   - Used: For executing tasks automatically (e.g., typing, driving)

---

| Memory Type     | Description               | When to Use                          |
|-----------------|---------------------------|--------------------------------------|
| Working         | Temporary task data       | Immediate decision-making            |
| Episodic        | Personal event storage    | Recalling specific experiences       |
| Semantic        | General knowledge storage | Understanding context and concepts   |
| Procedural      | Skill-based memory        | Automating routine tasks             |
