## Critical Issue: Mixed Authorship

This builder has a "two-generation" problem:
- **n03_rewrite ISOs (3)**: schema, quality_gate, output_template -- all 9.5, excellent
- **wave1_builder_gen ISOs (10)**: all below 7.5, several failing critical gates

The n03_rewrite set is consistent with OpenAI Realtime API v2024-12-17. The wave1_builder_gen
set treats realtime_session as generic WebRTC session management (video conferencing), not
LLM streaming. This mismatch is the root cause of most failures.
