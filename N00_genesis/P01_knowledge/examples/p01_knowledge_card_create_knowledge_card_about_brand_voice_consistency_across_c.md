# Brand Voice Consistency Across Communication Channels

## Quick Reference
```yaml
topic: brand_voice_consistency

criticality: high
```

## Key Concepts
- **Core Voice Elements**: 3-5 personality traits that remain constant (e.g., professional, approachable, innovative)
- **75% Consistency Rule**: Maintain 75% of core elements while adapting 25% for channel-specific needs
- **Voice Deviation Score**: Quantitative measure of consistency using semantic analysis tools
- **Channel Personas**: Platform-specific voice adaptations within brand framework
- **Tone Spectrum**: Formal to casual scale mapped per channel audience expectations

## Strategy Phases
1. **Audit**: Analyze existing content across channels using voice consistency scoring
2. **Define**: Document 3-5 core personality traits and create master voice guide
3. **Adapt**: Create channel-specific guidelines maintaining core brand DNA
4. **Implement**: Deploy training materials and approval workflows for content teams
5. **Monitor**: Monthly voice audits with consistency scores and deviation alerts

## Golden Rules
- MAINTAIN brand personality traits constant across all touchpoints
- ADAPT only tone, formality, and format while preserving core voice
- DOCUMENT specific vocabulary, phrases, and prohibited language per channel
- TRAIN content creators quarterly on voice application and channel nuances
