# Market Research Data Triangulation for Insight Validation

## Quick Reference

| Element | Detail |
|---------|--------|
| **Topic** | Data triangulation for market research |
| **Scope** | Validation across sources and methodologies |
| **Owner** | Market researchers, insights teams |
| **Criticality** | High - affects decision accuracy |

## Key Concepts

| Concept | Definition | Example | Bias Reduction |
|---------|------------|---------|----------------|
| **Data Triangulation** | 3+ independent sources validate same finding | Surveys + interviews + sales data | 60-80% |
| **Methodological Triangulation** | Mix quant/qual approaches | Statistics + ethnography + observation | 45-65% |
| **Investigator Triangulation** | Multiple researchers analyze same data | 2+ analysts review findings independently | 35-50% |
| **Convergence Threshold** | Sources agreeing within acceptable range | <15% variance indicates reliable insight | N/A |

## Strategy Phases

| Phase | Action | Key Output | Time Impact |
|-------|--------|------------|-------------|
| **Source Mapping** | Identify 3-5 independent data sources | Source validation matrix | +10% |
| **Method Mix** | Combine 2+ methodologies (quant + qual) | Methodology design | +20% |
| **Data Collection** | Execute parallel/sequential collection | Raw datasets | +40% |
| **Convergence Analysis** | Compare findings, flag >25% conflicts | Conflict report | +15% |
| **Synthesis** | Weight sources, resolve conflicts, document confidence | Validated insights | +25% |

## Golden Rules

| Rule | Rationale | Impact | Non-Compliance Risk |
|------|-----------|--------|-------------------|
| **Minimum 3 sources** | 2 sources = comparison, not triangulation | Validates reliability | False confidence |
| **Mix methods** | Pure quant or qual reduces triangulation power | Captures blind spots | Method bias |
| **Maintain independence** | Shared methodology = single source disguised | True validation | Pseudo-triangulation |
| **Document conflicts** | Disagreement often reveals hidden insights | Uncovers nuances | Missed opportunities |
| **Weight by quality** | Adjust influence by sample size and rigor | Accurate conclusions | Skewed results |

## Common Pitfalls
- **Pseudo-Independence**: Using same panel across "different" surveys
- **Method Bias**: Only mixing quant methods or only qual methods
- **Cherry-Picking**: Highlighting convergent data while ignoring conflicts
- **Late Triangulation**: Attempting validation after decisions made
- **Over-Weighting**: Letting one high-quality source dominate others
