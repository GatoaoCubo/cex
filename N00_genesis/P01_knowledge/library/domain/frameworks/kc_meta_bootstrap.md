# Meta-Bootstrap

## Bootstrap Sequence
```
SEED (minimal core, hand-crafted)
  → BUILDER-BUILDER (meta-factory, uses seed)
  → BUILDERS (generated by meta-factory)
  → ARTIFACTS (generated by builders)
  → SYSTEM COMPLETE (self-sustaining)
```

## CEX Bootstrap History
1. Hand-crafted: kinds_meta.json, 3 builder specs
2. Meta-factory: _builder-builder creates all builders
3. Builders: 301 builders produce artifacts
4. Self-sustaining: auto-evolve fills remaining gaps
5. Brand: /init personalizes for user's company

## Chicken-Egg Resolution
The meta-factory needs builder specs to know how to build.
But builder specs are what it produces.
Solution: seed the meta-factory with 13 meta-templates (bld_meta_*.md)
that define the STRUCTURE of builder specs, not the content.
