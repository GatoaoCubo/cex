Procedural memory is a type of long-term memory that encodes information about how to perform tasks and skills. It stores motor skills, reflexes, and habitual behaviors through neural pathways in the basal ganglia and cerebellum.

Key characteristics:
- Automatic execution of learned skills (e.g., riding a bike)
- Implicit memory - not consciously recalled
- Resilient to forgetting (more durable than declarative memory)
- Develops through repetition and practice
- Enables complex sequences like typing, playing instruments, or driving

Examples:
- Typing on a keyboard
- Riding a bicycle
- Swimming techniques
- Playing a musical instrument
- Writing with a pen

Distinguishes from declarative memory by:
1. Being unconscious rather than conscious
2. Involving motor patterns rather than facts
3. Showing greater resistance to decay over time
4. Requiring physical practice for retention
