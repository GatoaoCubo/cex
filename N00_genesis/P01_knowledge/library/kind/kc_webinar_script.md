# Webinar Script Format Guide

## Structure
1. **Intro (0:00-5:00)**
   - Welcome slide + host introduction
   - Webinar objective statement
   - Agenda preview

2. **Agenda (5:00-7:00)**
   - 3-5 key topics with time allocations
   - Interactive element preview
   - Q&A session reminder

3. **Segments (7:00-95:00)**
   - Topic 1: [Subheading]
     - Key points (3-5)
     - Visual cue: [Slide number]
   - Topic 2: [Subheading]
     - Key points (3-5)
     - Visual cue: [Slide number]
   - Topic 3: [Subheading]
     - Key points (3-5)
     - Visual cue: [Slide number]

4. **Q&A (95:00-105:00)**
   - Moderated questions
   - Live chat integration
   - Poll question: [Topic]

5. **CTA (105:00-110:00)**
   - Call to action statement
   - Registration link + deadline
   - Social media hashtag

## Speaker Notes
- Use bold for key terms
- Include timing cues
- Add visual references
- Note interactive elements

## Slide Cues
- [Slide 1] Title slide with date
- [Slide 2] Agenda overview
- [Slide 3] Topic 1 summary
- [Slide 4] Data visualization
- [Slide 5] Case study example
- [Slide 6] Interactive poll
- [Slide 7] Q&A instructions
- [Slide 8] Final CTA

### Customization Guide
1. Replace bracketed text with specific details
2. Use bold for key terms in speaker notes
3. Maintain 10-15 word count per slide title
4. Include 3-5 key points per topic
5. Add 2-3 interactive elements
6. Verify all timing cues
7. Include 1-2 polls/questions
8. Add social media integration
