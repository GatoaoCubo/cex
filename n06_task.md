The RO-Crate 1.2 Workflow Run Crate is a standardized container for scientific workflow execution provenance. It enables reproducible research by capturing metadata about workflow execution, including:

1. **CreateAction** - Records the workflow execution event with timestamps, inputs, and outputs
2. **ORCID Attribution** - Links workflows to researchers via ORCID identifiers
3. **FAIR Metadata** - Ensures Findable, Accessible, Interoperable, and Reusable data principles

Key features:
- Immutable execution provenance tracking
- Versioned workflow artifacts
- Cross-platform compatibility
- Semantic metadata for data discovery
- Audit trail for reproducibility

This crate format supports collaborative science by providing:
- Transparent workflow execution history
- Persistent attribution of intellectual contributions
- Standardized data packaging for reuse
- Machine-readable metadata for automated processing

Compliant with W3C PROV standards and extends RO-Crate 1.2 to include execution context metadata. Essential for reproducible computational research and FAIR data principles implementation.
