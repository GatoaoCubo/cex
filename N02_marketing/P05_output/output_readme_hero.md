# CEX — Typed Knowledge System for LLM Agents

## Your AI agents ship half-baked outputs. CEX makes every artifact pass 8 mandatory gates before it exists.

![MIT License](https://img.shields.io/badge/License-MIT-blue.svg)

### The Problem

LLM agents produce inconsistent, unstructured outputs. No validation. No schema enforcement. No quality floor. You're shipping vibes, not artifacts.

### The Solution

CEX transforms LLM chaos into deterministic agent workflows through **typed knowledge schemas** and **mandatory quality validation**. Every artifact passes through an 8-stage pipeline (8F) that enforces structure, validates output, compiles to YAML, and blocks anything below the quality threshold.

### What You Get

| Capability | Details |
|-----------|---------|
| **8F Pipeline** | Frame → Filter → Fan → Focus → Forge → Format → Feedback → File |
| **123 Typed Kinds** | Every artifact has a schema. `email_sequence`, `knowledge_card`, `landing_page` — all constrained. |
| **8 Specialized Nuclei** | N01 researches. N02 writes copy. N03 builds. N04 documents. N05 codes. N06 monetizes. N07 orchestrates. |
| **Quality Gates** | Nothing ships below 8.0. Peer review is mandatory. Self-scoring is forbidden. |
| **GDP Protocol** | User decides WHAT (tone, audience, style). LLM decides HOW (files, pipeline, structure). |
| **Memory System** | 4-type taxonomy with age decay. Corrections, preferences, conventions, context — all tracked. |

### Quick Start

```bash
git clone https://github.com/yourusername/cex.git
cd cex
pip install -r requirements.txt
python _tools/cex_bootstrap.py
```

Then open your LLM agent and type `/init` to configure brand identity.

### Architecture at a Glance

```
User Intent
  └→ 8F Pipeline (Frame → File)
       ├→ N01 Research    ├→ N04 Knowledge
       ├→ N02 Marketing   ├→ N05 Operations
       ├→ N03 Builder     ├→ N06 Strategy
       └→ N07 Orchestrator (dispatches all)
            └→ Quality Gate ≥ 8.0 → Ship
```

### Who Uses This

- **Solo developers** building AI-powered products who want structured agent output
- **AI teams** needing reproducible, quality-gated artifact pipelines
- **Content operations** requiring multi-format, brand-consistent generation

### Core Philosophy

> Quality is not optional. Structure is not overhead. Every shortcut you skip today becomes technical debt tomorrow.

CEX enforces this through typed schemas (123 kinds), mandatory frontmatter, compilation gates, and peer-review scoring. The system trusts no single nucleus to judge its own work.
