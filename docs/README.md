# CEXAI Documentation

CEXAI (Cognitive Exchange AI) is a typed knowledge system for LLM agents. It organizes AI work into 300 artifact types, 12 domain pillars, and 7 specialized nuclei -- all driven by an 8-function reasoning pipeline. The X in CEXAI stands for Exchange: intelligence compounds when shared. Every typed artifact is a portable exchange unit -- import it into any CEXAI instance, any runtime (Claude, GPT, Gemini, Ollama), and it validates automatically.
